# Product Catalog Microservice
An enterprise-grade e-commerce microservice built with **Java 21** and **Spring Boot**, featuring a containerized **PostgreSQL** database and automated schema migrations.

---

## 🚀 Features
* **Full CRUD Operations:** Manage products with RESTful endpoints.
* **Database Versioning:** Automated schema migrations using **Flyway**.
* **Containerized Environment:** Fully orchestrated with **Docker** and **Docker Compose**.
* **API Documentation:** Interactive documentation via **Swagger UI / OpenAPI 3.0**.
* **Quality Assurance:** Unit testing with **JUnit 5** and **Mockito** (80%+ coverage).

---

## 🛠 Tech Stack
* **Backend:** Java 21, Spring Boot 4.0.2, Spring Data JPA
* **Database:** PostgreSQL 15
* **Infrastructure:** Docker, Docker Compose
* **Migration:** Flyway
* **Testing:** JUnit 5, Mockito, AssertJ
* **Documentation:** SpringDoc OpenAPI (Swagger)
